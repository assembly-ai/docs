## Authentication

Endpoints documented in the OpenAPI specification below use Bearer authentication as defined in that spec. Other guides (for example [Interactions](/usage/developer/interactionsAPI)) may use a different header format—follow each page’s examples.

```json
"security": [
  {
    "bearerAuth": []
  }
]
```
